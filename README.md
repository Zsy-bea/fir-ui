# fir-ui

## 安装组件库
```
npm add fir-lib-ui
```

### 全局导入
```
import FirUI from 'fir-lib-ui'
import 'fir-lib-ui/dist/fir-ui.css'
Vue.use(FirUI)
```

### 使用
```
<fir-button type="danger" circle>拆</fir-button>
```
